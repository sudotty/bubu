## 技术规范

### 代码规范
```typescript
// 组件命名规范
- 组件名使用PascalCase: DataTable, TableHeader
- Hook名使用camelCase: useTableState, useTableSort
- 类型名使用PascalCase + 后缀: TableProps, SortConfig

// 文件组织规范
- 每个组件一个文件夹
- index.ts作为导出入口
- types.ts集中类型定义
- hooks/目录存放相关Hook
```

### 性能要求
```typescript
// 性能基准
- 1000行数据渲染时间 < 100ms
- 10000行数据虚拟滚动流畅度 > 60fps
- 组件包体积 < 50KB (gzipped)
- 首次渲染时间 < 200ms
```

### 兼容性要求
```typescript
// 浏览器支持
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

// React版本
- React 19.x
- TypeScript 5.x
```
