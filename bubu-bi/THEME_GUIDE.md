# bubu-bi 主题系统指南

## 概述

bubu-bi 现已升级到 daisyUI 5.0，支持多主题切换，包括专业的金色主题系统。

## 可用主题

### 1. 金色主题 (golden) - 默认推荐
- **设计理念**: 专业、温暖、高端
- **主色调**: 温暖的金色 (oklch(65% 0.15 85))
- **适用场景**: 商务演示、专业分析、品牌展示
- **特点**: 提升产品专业感和品牌价值

### 2. 深色金主题 (golden-dark)
- **设计理念**: 优雅、专业、护眼
- **主色调**: 更亮的金色配深色背景
- **适用场景**: 长时间使用、夜间工作
- **特点**: 减少眼部疲劳，保持专业感

### 3. 浅色主题 (light)
- **设计理念**: 简洁、清爽、传统
- **适用场景**: 日间办公、打印输出
- **特点**: 高对比度，适合阅读

### 4. 深色主题 (dark)
- **设计理念**: 现代、酷炫、护眼
- **适用场景**: 开发环境、夜间使用
- **特点**: 减少蓝光，保护视力

### 5. 绿色主题 (green)
- **设计理念**: 自然、舒适、专业
- **主色调**: 专业森林绿 (oklch(55% 0.15 140))
- **适用场景**: 数据分析、长时间工作、自然偏好用户
- **特点**: 减少视觉疲劳，提升专注力

### 6. 深色绿主题 (green-dark)
- **设计理念**: 自然、护眼、高端
- **主色调**: 更亮的绿色配深色背景
- **适用场景**: 夜间工作、护眼需求、绿色偏好用户
- **特点**: 结合绿色的舒适感和深色的护眼特性

## 主题切换

### 用户界面切换
1. 点击顶部工具栏右侧的主题选择器
2. 从下拉菜单中选择所需主题
3. 主题会立即应用并自动保存

### 程序化切换
```javascript
// 设置主题
document.documentElement.setAttribute('data-theme', 'golden');
localStorage.setItem('theme', 'golden');
```

## 开发指南

### 添加新主题
1. 在 `src/style.css` 中添加新的 `@plugin "daisyui/theme"` 块
2. 在 `tailwind.config.js` 中的 `themes` 数组添加主题名
3. 在 `ThemeSelector.tsx` 中添加主题选项

### 组件开发最佳实践
1. **使用语义化颜色类**: `text-primary` 而非 `text-yellow-500`
2. **支持主题切换**: 避免硬编码颜色值
3. **测试所有主题**: 确保在所有主题下都能正常显示
4. **遵循设计系统**: 使用统一的颜色变量

## 故障排除

### 主题不生效
1. 检查 `data-theme` 属性是否正确设置
2. 确认 CSS 文件已正确导入
3. 清除浏览器缓存

### 颜色显示异常
1. 检查是否使用了硬编码颜色
2. 确认 OKLCH 语法正确
3. 验证 CSS 变量定义

### 性能问题
1. daisyUI 5.0 已大幅优化性能
2. 使用语义化类名减少 CSS 体积
3. 避免过度自定义样式
