# README

## bubucel - 智能数据分析工具

## 产品简介

bubucel 是一个基于火山引擎LLM的智能数据分析工具，专注于提供自然语言到SQL的转换能力，以及组件化的数据查询管理。通过极简的MVP设计，让数据分析变得更加智能和高效。

## 核心特性

### 🤖 智能查询引擎
- **自然语言查询**：基于火山引擎LLM API，支持中文自然语言转SQL
- **SQL查询模式**：保留传统SQL编写方式，满足高级用户需求
- **双模式切换**：自然语言（默认）和SQL（高级）模式可自由切换
- **AI执行策略**：生成SQL后用户确认执行，确保安全性

### 📦 组件库管理
- **业务组件**：保存成功执行的查询为可复用组件
- **Excel组件**：预留Excel数据处理组件功能
- **智能排序**：按执行时间倒序，最新组件在上
- **一键复用**：选择组件自动填充查询内容

### 🛡️ 安全与易用
- **DDL操作提醒**：自动识别数据定义语言操作并提醒用户
- **错误重试机制**：LLM格式错误自动重试，连续3次失败后报错
- **标准化输出**：LLM返回固定JSON格式（业务ID/SQL/定义/描述）

## 技术架构

### 前端技术栈
- **框架**：React + TypeScript
- **样式**：Tailwind CSS + DaisyUI
- **构建**：Vite + Wails

### 后端技术栈
- **语言**：Go
- **LLM集成**：火山引擎LLM API
- **数据库**：SQLite

### 核心原则
1. **火山引擎LLM智能处理**：真实API集成，非Mock数据
2. **极简MVP设计**：专注核心功能，避免过度设计
3. **组件库管理**：查询结果组件化，提高复用效率
4. **P2P分享预留**：为未来功能扩展预留架构空间
5. **技术架构原则**：现代化技术栈，确保可维护性

## 快速开始

### 环境要求
- Go 1.19+
- Node.js 16+
- Wails v2

### 安装依赖
```bash
# 安装前端依赖
cd frontend
npm install

# 返回项目根目录
cd ..
```

### 开发模式
```bash
# 启动开发服务器
wails dev
```

开发服务器将启动Vite热重载服务器，同时在 http://localhost:34115 提供浏览器开发环境，可以在浏览器开发工具中调用Go方法。

### 生产构建
```bash
# 构建生产版本
wails build
```

## 使用指南

### 自然语言查询
1. 在查询输入框中输入中文描述
2. 点击执行按钮，AI将生成对应SQL
3. 确认AI解析结果后执行查询
4. 查看结果并可选择保存到组件库

### 组件库使用
1. 点击"组件库"按钮打开面板
2. 选择"业务组件"或"Excel组件"
3. 点击目标组件自动填充查询
4. 执行查询获得结果
